Request Flow

User types → POST /api/chat
→ API route sends message + tools + system prompt to OpenAI/Gemini
→ LLM replies with tool_call (e.g. search_hotels)
→ API route executes the tool function → gets results
→ Sends results back to LLM
→ LLM replies with structured JSON:
{ message: "...", itinerary_update: {...}, options: [...] }
→ API route returns JSON to frontend
Frontend receives JSON:
→ message → renders chat bubble
→ itinerary_update → updates itinerary panel + drops map pin
→ options → populates Options tab cards

Free APIs Used (no scraping, no ToS issues)

Open-Meteo
Weather by city + month. Completely free. No API key needed.
OpenTripMap
Attractions & POIs. Free tier. Returns coordinates for map pins.
Google Places API
Hotels & restaurants. Free tier ($200 credit/month — enough for a capstone).
OSRM (routing)
Route drawing on map between locations. Free public instance.
OpenStreetMap
Map tiles via Leaflet.js. Free forever. No API key.
estimate_budget
Pure JS calculation — no API. Uses prices returned from other tools.
